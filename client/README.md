# client

MPP client sample using Bun, mppx, and the Tempo payment method.

To install dependencies:

```bash
bun install
```

Set a Tempo private key before running:

```bash
export TEMPO_PRIVATE_KEY=0x...
```

Run the client:

```bash
bun run index.ts
```
